Draw a matplotlib plot of the permutation. Can be used for both quick visualization, or to build a larger figure. Unrecognized arguments are passed as options to the axes object to allow for customization (i.e., setting a figure title, or setting labels on the axes). Falls back to an ascii_plot if matplotlib isn’t found, or if use_mpl is set to False.
