classmethod random_avoider(cls, n, B, simple=False, involution=False, verbose=- 1)[source]

Generate a (uniformly) random permutation that avoids the patterns contained in B.

Parameters

  • n (int) – length of permutation to generate

  • B (iterable) – Iterable of permutation-like objects to avoid.

  • simple (Boolean, optional) – Whether the returned Permutation should be simple. Defaults to False.

  • involution (Boolean, optional) – Whether the returned Permutation should be an involution. Defaults to False.

  • verbose (int, optional) – Level of verbosity (-1 for no verbosity) Doubling the integer doubles the number of messages printed. Defaults to -1.

Returns

A permutation avoiding all the patterns in B

Return type

p (Permutation)

Examples

>>> p = Permutation.random_avoider(8, [123])
>>> p.involves(123)
False
